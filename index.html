HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language and is the standard markup language for creating Web pages.
  • Consists of a series of elements that tell the browser how to display content.

CSS

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit takes the staged snapshot and commits it to the project history.
  • git commit along with git add defines the basic workflow for Git users.

JavaScript

  • The world's most popular programming language.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript function declaration tells the compiler about a function's name, return type and parameters set. A function definition provides the actual body of the function.
